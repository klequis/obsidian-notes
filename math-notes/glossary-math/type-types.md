# Type
#type #adt #data-type

- A collection of similar objects
- Objects can be fundamental, pointers, or have their type determined by their class. 

## ADT

- "Abstract Data Type"
- A data type with hidden implementation details

### Data Type
- A collection of values 
- together with the operations that use them and produce them
- plus the assumptions that can be made about the operations and values. 

source: [Glossary of Computer Programming Terms](http://csci.csusb.edu/dick/cs202/glossary.html)
